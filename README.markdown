# jQuery Time

Usage

`<time datetime="" data-display="countdown"></time>`

or

$('time').countdown();

# Display Reference
%s  : Seconds
%ss : Seconds Padded
%m  : Minutes
%mm : Minutes Padded
%h  : Hours
%hh : Hours Padded
%d  : Days
%dd : Days Padded

%SECONDS : Second(s)
%MINUTES : Minute(s)
%HOURS   : Hour(s)
%DAYS    : Day(s)
